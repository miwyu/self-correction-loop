# README rubric for wordfreq

The README is done when every criterion below holds.

1. README.md has a Requirements section that states the minimum Python
   version and that no third-party packages are needed.
2. README.md documents every flag shown by `python wordfreq.py --help`,
   including each flag's default value where one exists.
3. README.md contains at least one complete usage example: the exact command
   run against the included `sample.txt`, followed by the output that command
   actually prints (the output in the README must match reality).
4. README.md documents the JSON output mode with an example of its real
   output.
5. README.md is under 120 lines.
6. `wordfreq.py` has not been modified.
7. Every command shown in README.md runs without error from the project
   directory.
